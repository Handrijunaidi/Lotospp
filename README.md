Lotos2
======

Lotos2 is a telnet-based talker server, written in C++.

If you don't know what a talker is, read http://en.wikipedia.org/wiki/Talker .


Dependencies
============

* cmake
* Lotos2 has several dependencies:
  * Boost libraries
   - boost
   - boost-program-options
   - boost-system
   - boost-filesystem
   - boost_thread
   - boost_date_time
   - boost_random
  * MySQL
* make


Build
=====

	cmake . && make
